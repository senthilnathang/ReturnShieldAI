# Docs

This folder contains:
- architecture notes
- roadmap updates
- developer workflow guidance
- demo script and FAQ
- pitch deck assets
- test case reference for the ML layer

## Key Files

- `architecture.md` - system architecture and data flow
- `roadmap.md` - delivery status and next work
- `developer-guide.md` - setup and contribution workflow
- `demo-script.md` - live demo narration and steps
- `faq.md` - common product and runbook questions
- `test-cases.md` - ML and backend test coverage summary
- `pitchdeck.md` - brochure content
- `pitchdeck-slideshow.html` - slide deck presentation
- `pitchdeck-bifold.html` - folded brochure layout
